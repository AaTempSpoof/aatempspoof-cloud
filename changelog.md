# AaTempSpoof v14.5 更新公告

## WebUI

- 优化纯白、纯黑主题切换，纯黑模式采用扁平化显示，去除卡片、按钮、输入框和底栏的多余材质效果。
- 调整黑色模式下的白色描边、圆角过渡、分类标签、状态文字和页面切换按钮，提高可读性与整体一致性。
- 修复开关光圈裁切、开关位置偏移以及底栏滑块描边过粗等显示问题。
- 修复「极致伪装」保存后重新进入页面自动关闭的问题。
- 「极致伪装」现在严格受「其他伪装」分类开关控制，状态提示与实际配置保持一致。
- 温度滑块的小数值支持点击后直接输入，仍保留滑块调节，并加强数值范围校验。
- 优化配置读取、保存和刷新后的状态同步，减少页面显示与实际配置不一致的情况。
- 优化检查更新页面和更新日志读取状态，补充失败提示和加载状态。

## 云端管理

- 赞助到账监听器增加独立通知日志，记录应用包名、标题、正文、ticker 和通知时间。
- WebUI 增加「通知日志」按钮，点击后通过弹窗查看所有应用通知；只有微信支付通知会触发赞助处理。
- 一次性注册链接发送成功后，自动向管理员邮箱发送通知，显示收件邮箱和实际赞助金额。
- 通知调试日志单独保存并自动轮转，避免长期运行导致日志无限增长。
- 自动审核改为串行、原子更新名单与签名，并在写入前建立待发布标记，避免并发审核或后台同步覆盖新用户。
- 公共仓库和私有仓库按清单及 SHA-256 完整同步，文件未补全前不开放 WebUI；断网时仅使用已验证缓存。
- GitHub 发布失败后持久保留任务并自动重试，重启后继续发布，不再丢失已通过但尚未上传的审核记录。

## 打包与兼容

- WebUI 继续使用加密封装发布，源码目录保留可维护版本，模块包内仅放入加密后的 WebUI。
- 构建脚本支持自动识别最高版本目录，并统一构建快捷开关 APK、主二进制、加密 WebUI 与根目录模块 ZIP，避免包内文件落后于源码。
- 修复关闭电池伪装后 BatteryService 测试状态未及时释放造成的电量不刷新、快充提示异常及短暂温度为 0；真实 0% 电量现在也能正常同步。
- 热控配置改为通过模块私有文件执行只读绑定挂载，不再直接改写系统、vendor 或 odm 分区，也不再删除厂商原始热控配置。
- 授权异常或完整性校验失败时改为安全停用，不再自动执行卸载和删除配置；授权恢复后仍可正常恢复运行。
- 优化守护进程 PID 身份校验、升级迁移和卸载清理，避免误杀同名进程、执行旧模块脚本或重复重启主进程。
